## Rubberduck

* code metrics 
* refactor 
* `free` - last release 2023, repository contains updates not released.

homepage - <https://github.com/rubberduck-vba/Rubberduck>

>-Refactor, all options including Rename **are greyed out**.  
>-Goto Rubberduck toolbar in the IDE, the left-most label is actually a button, says `Pending` at first, but then you click it and it goes through a number of states before it stops at `Ready` - at that point everything should be enabled everywhere. [source](https://github.com/rubberduck-vba/Rubberduck/discussions/6104)
